THIS IS A SMALL APPLICATION THAT HELPS YOU UNDERSTAND 
THE WIKIMEDIA ENTERPRISE API 

IT ALLOWS YOU TO:
1. AUTHENTICATE (LOGIN) USING THE WIKIMEDIA API
2. FILTER WIKIMEDIA PROJECTS BY LANGUAGE 
3. SEARCH FOR A SPECIFIC ARTICLE USING THE WIKIVOYAGE ENDPOINT 


TECHNOLOGIES:
- JavaScript 
- Vue Js (babel, router, Axios)
- Bootstrap 
- Netlify - for hosting
